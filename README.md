Step by step project development
# Cloning the project

1. Fork the project
2. Clone the project using git clone https://github.com/MbuyiselwaH/Facility-Booking-System-Group40-Project.git
3. Create the virtual environment called Venv (Or any other prefered name)
4. Install requirements.txt
5. Run the project
6. Continue to build the project!!!
